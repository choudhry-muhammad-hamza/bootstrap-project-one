# Bootstrap Project One 🚀

This is a simple responsive website built using **HTML and Bootstrap**.  
The project is created for practice and to improve frontend development skills.

---

## 📌 Features

- Responsive navbar using Bootstrap
- Carousel section with captions
- Product cards layout
- Dropdown menu functionality
- Login and signup modal forms
- Contact section with structured layout
- Fully mobile-friendly responsive design

---

## 🛠️ Technologies Used

- HTML5
- Bootstrap 5

---

## 📂 Project Structure

- index.html
- about.html
- contact.html

---

## 🚀 How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Explore different pages

---

## 🎯 Purpose

This project was built for learning Bootstrap and improving frontend development skills by creating a real-world website layout.
